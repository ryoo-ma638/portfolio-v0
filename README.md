# ポートフォリオ v0

愛知工業大学の授業「Webデザイン」の最終課題として制作した、初めての自作ポートフォリオサイトです。HTML / CSS / JavaScript だけで実装しています。

🔗 **公開サイト** … <https://ryoo-ma638.github.io/portfolio-v0/>

タイトルの **v0** には、このサイトを自分のポートフォリオの「原点（最初の版）」と位置づけ、これから v1・v2 と作り直しながら育てていく、という意味を込めています。

## ページ構成

| ページ | パス | 内容 |
| --- | --- | --- |
| トップ | `index.html` | ヒーロー・プロフィール・作品・経歴・推し紹介をまとめた入口 |
| プロフィール | `profile/` | 自己紹介の詳細 |
| 作品 | `works/` | 制作物の紹介 |
| 経歴 | `memory/` | これまでの歩み |
| 推し紹介 | `yumiki/` | 応援している推しの紹介 |
| ローディング | `loading.html` | ページ遷移時の演出画面 |

## 主な機能

- ローディングアニメーション付きのページ遷移
- ダーク / ライトのテーマ切り替え
- スクロールに連動したフェードイン表示
- レスポンシブ対応（スマホ用のオフキャンバスメニュー）
- 作品をクリックで開くモーダル表示

## 使用技術

- HTML5 / CSS3 / JavaScript（フレームワークは使用していません）
- Google Fonts（Noto Sans JP / Poppins）

## ローカルでの表示方法

```bash
git clone https://github.com/ryoo-ma638/portfolio-v0.git
cd portfolio-v0
```

クローン後、`index.html` をブラウザで開くと表示できます。
